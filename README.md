# hiproxy-plugin-dashboard
hiproxy dashboard plugin to manage your projects config and services

## Usage

### install

```bash
npm install hiproxy/hiproxy-plugin-dashboard -g
```

### start server
```bash
hiproxy start
```

### visit

<http://127.0.0.1:5525/dashboard/>

### Development

```bash
npm install
webpack -w
```

### Bundle

```
webpack
```
